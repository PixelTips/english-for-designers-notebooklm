# English Mastery for Global Design Careers 🌍🎨

Este projeto foi desenvolvido como um **Caderno Temático no NotebookLM** para consolidar o aprendizado de inglês técnico voltado para o mercado de Design Global. O foco é unir competência linguística e estratégia profissional para atuar em empresas estrangeiras.

## 🎯 Objetivos
* Centralizar termos técnicos de Product e Graphic Design.
* Dominar as "32 conexões" da metodologia Gabi Modesto para formação de frases.
* Criar um repositório de consultas rápidas para entrevistas e apresentações de portfólio.

## 📚 Curadoria de Fontes
Utilizei uma seleção estratégica de fontes para alimentar a IA:
* **Canais YouTube:** EnglishBay, Gabi Modesto, Kultivi.
* **Técnico:** Relatórios de Deep Research sobre Design e Product Design.
* **Metodologia:** Guia prático das 32 conexões linguísticas.

## 🤖 Engenharia de Prompts e Insights
Durante o processo, testei prompts para extrair o melhor da IA:
1. **Prompt:** "Quais termos em inglês são essenciais para designers de produto?"
   - *Resultado:* A IA listou termos como *User Journey, Stakeholders e Handoff*.
2. **Prompt:** "Simule uma pergunta de entrevista sobre meu processo criativo."
   - *Ajuste:* Percebi que detalhar o contexto de "Product Design" gera respostas muito mais úteis do que "Design" genérico.

## 📖 Miniguia de Estudos
* **Glossário Base:** *Scalability, Pain points, Accessibility, Branding Strategy.*
* **Dica de Ouro:** Utilizar o "Audio Overview" do NotebookLM para treinar o listening com sotaques nativos.
* **Próximo Passo:** Praticar a fluência para entrevistas em empresas como a Meta.
